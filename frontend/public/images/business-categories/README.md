# Business Category Images

Place your business category images in this folder.

## Required Images

Add the following images with these exact names:

1. **restaurants.jpg** - Image for Restaurants and Dining category
2. **hotels.jpg** - Image for Hotels category
3. **retail.jpg** - Image for Retail category
4. **salons.jpg** - Image for Salons & Spas category
5. **fitness.jpg** - Image for Fitness Centers category
6. **seasonal.jpg** - Image for Seasonal Gifting category

## Image Specifications

- **Format**: JPG, JPEG, PNG, or WebP
- **Recommended Size**: 600x400 pixels (3:2 aspect ratio)
- **File Size**: Optimize images to keep file sizes under 200KB for better performance
- **Quality**: Use high-quality images that represent each business category

## Fallback

If an image is not found or fails to load, the card will automatically fall back to a gradient background with the category's color scheme.

## Usage

Images are automatically loaded from this folder when the landing page displays business category cards. No code changes needed - just add the images with the correct filenames.
